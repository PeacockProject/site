Supported Targets

We are actively building kernels, firmware, and UI ports for the following hardware targets. Every test helps push the project forward.

Oppo A16

Active target. Includes MTK boot implementations (MinKernel) and deep display/framebuffer fixes.

Samsung Galaxy S4 (jflte)

Legacy revival. Focus on fbdev-compat layers, SDDM quirks, and custom firmware delivery.

Xiaomi Mi A2 Lite (daisy)

Active target. Includes tailored PRP (Peacock Recovery Project) and remote network debugging (netssh) integrations.

QEMU (x86_64)

Virtual target. Used heavily for host building validation, UI testing, and core OS environment verification without hardware.
